# Online Community Working Group

## Our Charter

Can be found [here](https://github.com/django/dsf-working-groups/blob/main/active/online-community.md)

## Meetings

We have quarterly meetings, minutes and agendas will be documented in the [meetings directory](meetings/)

## Ideas

We are open to any and all ideas to improve the community experience online. To begin with we will be setting up a project and issues to track them. They will be reviewed periodically.
If you are particularly keen on an idea being implemented then we will welcome your involvement (assuming permissions on relevant platforms allow this)
